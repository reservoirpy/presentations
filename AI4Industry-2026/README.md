**# IA4industry 2026 - Tutoriel sur ReservoirPy**

Vous trouverez dans ce dépôt la présentation sur le Reservoir Computing lors de la semaine du [Workshop AI4industry](https://ai4industry.fr/) à Bordeaux (FR). Nous fournissons également différents tutoriels pour démarrer rapidement avec notre bibliothèque ReservoirPy.

1. Voici les [diapositives de la présentation](Reservoir-Computing_AI4industry2026.pdf) sur le Reservoir Computing faite le mardi 20 janvier 2026 à l'ENSEIRB-MatMeca, Bordeaux.

2. Pour tester ReservoirPy facilement ouvrez un terminal, Juputer Notebook ou Google Colab et installez ReservoirPy :
```bash
pip install reservoirpy[hyper]
```

3. Lancez le [tutoriel court en français](Introduction-courte-en-francais-avec-exercices.ipynb ) avec une analyse de données d'électro-encpéphalogramme (EEG) et des exercices à compléter.

4. Lancez un [tutoriel des fonctionalités avancées](Advanced-introduction-in-English.ipynb) de ReservoirPy en Anglais incluant les recherches d'hyperparamètres (HP) dynamique et non-dynamiques. Vous pouvez suivre sur les [diapos de présentation](Reservoir-Computing_AI4industry2026.pdf) à partir de la page 130 environ. (Si vous téléchargez le code sur votre ordinateur, pensez à bien télécharger tout le dossier afin de récupérer également le fichier 'interactive_report.py' pour la recherche d'HP dynamique.)

5. Vous pouvez également trouver des informations complémentaires sur ReservoirPy en Français dans les [diapos de la présentation dédiée à ReservoirPy](ReservoirPy-AI4industry2025_ReservoirPy-0.3.pdf) (lors d'AI4industry 2025). (Notes que cela concerne une version précédente de ReservoirPy (la v0.3.*), il se peut que certaines fonctions aient changées.)

6. Pour aller plus loin et voir toutes les fonctionalités de ReservoirPy et en apprendre plus sur le Reservoir Computing, allez voir :
- [la page principale de la bibliothèque Reservoirpy](https://github.com/reservoirpy/reservoirpy) 
- [la page des tous les tutoriels](https://github.com/reservoirpy/reservoirpy/tree/master/tutorials) sous forme de Notebook Jupyter
  - Pour ouvrir directement le [premier tutoriel dans Google Colab](https://colab.research.google.com/github/reservoirpy/reservoirpy/blob/master/tutorials/1-Getting_Started.ipynb)
- [la documentation officielle](https://reservoirpy.readthedocs.io/) ReadTheDocs
  - Incluant le [guide utilisateur](https://reservoirpy.readthedocs.io/en/latest/user_guide/index.html#user-guide) sous forme de pages web.


N'oubliez pas de mettre une ⭐️ à ce tutoriel et à la bibliothèque si vous trouvez ça utile afin de les faire connaître au plus grand nombre !
